\section{Características}
\begin{itemize}
\item Tipo de trabajo: Articulo de investigación aplicando métodos de minería de datos 
\begin{itemize}
\item El proyecto puede ser grupal de hasta 3 personas
\item Debe trabajar sobre una base de datos nacional
\item La temática de la base de datos es a elección del estudiante
\item Se debe incluir los métodos vistos en la materia o alguno relacionado
\item Bases de datos con al menos 10 mil observaciones
\end{itemize}
\item Los trabajos no deben exceder las 15 páginas (sin contar anexos y caratula)
\item Al final se debe preparar una presentación del trabajo, (máximo 10 minutos)
\item El artículo debe escribirse en Latex/Sweave/rmarkdown
\item El trabajo debe contar con un repositorio en GitHub
\item Entregables en repositorio de GitHub:
\begin{itemize}
\item Código de R
\item Articulo de investigación
\item Presentación
\end{itemize}
\end{itemize}

\section{Contenido mínimo del articulo de trabajo}
El contenido mínimo es:
\begin{itemize}
\item \textbf{Introducción}
\item \textbf{Objetivos}
\item \textbf{Motivación}
\item \textbf{Marco Teórico / Revisión de literatura} 
\item \textbf{Descripción de la base de datos}
\item \textbf{Metodología}
\item \textbf{Resultados y análisis}
\item \textbf{Conclusiones y recomendaciones}
\item \textbf{Referencias}
\end{itemize}
