\section{Características}
\begin{itemize}
\item Tipo de trabajo: Dashboard (Panel de información)
\begin{itemize}
\item Debe trabajar sobre una base de datos nacional o internacional (que incluya a Bolivia)
\item La temática de la base de datos es a elección del estudiante
\item Se debe implementar al menos 1 método de minería de datos visto en la materia
\item Bases de datos con al menos 10 mil observaciones y 10 variables
\item El estilo y contenido de la visualización de la información en la plataforma depende del estudiante
\end{itemize}
\item El trabajo puede realizarse en grupos de máximo 3 personas o personal.
\item Se debe incluir un reporte de investigación que documente la realización del dashboard. El reporte no debe exceder las 10 páginas (sin contar anexos y caratula)
\item Al final se debe preparar una presentación del trabajo, (máximo 10 minutos)
\item El reporte del trabajo  debe estar en Latex o Rmarkdown
\item Entregables
\begin{itemize}
\item Código de R del dashboard alojado en un repositorio en Github
\item Reporte del trabajo
\end{itemize}
\end{itemize}
